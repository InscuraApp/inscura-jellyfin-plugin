# Inscura 用 Jellyfin メタデータプラグイン

言語: [English](../README.md) | [简体中文](README-zh.md) | **日本語** | [한국어](README-ko.md)

Inscura は、映画情報、キャスト、ジャンル、ジャケット、背景画像、予告編などのデータを整理できるローカルメディアライブラリアプリです。Jellyfin はメディアライブラリの再生と管理を担当しますが、Inscura で整理されたこれらのデータについては認識していません。

このプラグインは、InscuraのローカルAPIサービスをJellyfinに連携させるために使用されます。Jellyfinが映画のメタデータを更新する際、プラグインはタイトル、あらすじ、公開日、評価、ジャンル、制作会社、キャスト、カバー画像、背景画像、サムネイルなどのデータをJellyfinに書き込みます。

このプラグインは、Inscuraのメディアライブラリデータとライブラリ内で生成された画像リソースのみを読み取り、元のメディアファイルのダウンロード、移動、名前変更、削除、または変更は行いません。

## 現在の機能

- Jellyfinから渡された実際のファイルパスとファイル名を優先して照合し、Jellyfinのタイトルが手動で変更されたことによる誤った照合を減らします。
- パスによる照合が見つからない場合、ファイル名に含まれる識別コードとJellyfinのタイトルによる照合を継続します。
- Jellyfinが受け付け可能な映画メタデータフィールド（タイトル、オリジナルタイトル、概要、公開日、年、評価、ジャンル、制作会社、国、タグ、キャスト、監督、脚本家、プロデューサー、ポスター、背景画像、サムネイルなど）への書き込みに対応しています。
- Inscura からローカルの画像リソースをインポートし、Jellyfin の Primary、Backdrop、Thumb、Banner、Logo、Art、Disc 画像の候補として使用できます。
- リモート予告編については、現在 YouTube の URL のみインポート可能です。**YouTube以外のローカルまたはオンライン予告編については、InscuraアプリのNFO機能でエクスポートした後、Jellyfinでスキャンしてください。**

## Inscuraのローカルインターフェースサービスの有効化

1. Inscuraを起動し、Jellyfinと同期したいメディアライブラリを開きます。
2. 設定内の「API サービス設定」に進み、ローカルインターフェースサービスを有効にします。
3. インターフェースサービスがトークン認証を使用している場合は、設定ページに表示されるインターフェーストークンを保存してください。
4. Jellyfin サーバーが稼働しているデバイスからヘルスチェックアドレスにアクセスし、サービスに接続できることを確認します。

例：

```bash
curl "http://[ip]:28687/api/v1/health"
```

Jellyfin と Inscura が同じマシン上にない場合、プラグインのサービスアドレスには「`127.0.0.1`」と入力せず、Inscura を実行しているコンピュータの LAN 内アドレスを入力してください。例：

```text
http://[ip]:28687
```

ローカル API サービスは、現在のメディアライブラリのライフサイクルに合わせて動作します。メディアライブラリが開いており、サービスが有効になっている間はポートをリッスンし、**メディアライブラリがロックされたり、閉じられたり、アプリケーションが終了したりすると、サービスは停止します。**

## 推奨インストール方法：プラグインリポジトリ

この方法では、Jellyfin がプラグインリポジトリから `manifest.json` を読み取るため、インストールやその後のアップグレードがより簡単になります。中国本土のネットワーク環境では、jsDelivr のアドレスを使用することをお勧めします。

リポジトリ名：

```text
Inscura
```

リポジトリ URL：

```text
https://cdn.jsdelivr.net/gh/InscuraApp/inscura-jellyfin-plugin@release/manifest.json
```

インストール手順：

1. Jellyfin コンソールにアクセスします。
2. プラグインディレクトリまたはプラグイン設定ページを開きます。
3. 「リポジトリ」または「Repositories」に移動します。
4. 「追加」をクリックします。
5. 名称に「`Inscura`」と入力します。
6. URLに上記のjsDelivrマニフェストのURLを入力します。
7. 保存後、プラグインカタログまたは「Catalog」に移動します。
8. 「`Inscura`」を見つけてインストールします。
9. Jellyfinを再起動します。

GitHubに安定してアクセスできる環境であれば、releaseブランチにあるmanifestファイルを使用することも可能です。ただし、中国本土の環境では、上記のjsDelivrのURLを優先して使用してください。

## 代替インストール方法：zip ファイルを直接ダウンロード

Jellyfin のプラグインリポジトリからインストールできない場合は、プラグインの zip ファイルを手動でダウンロードし、Jellyfin のプラグインディレクトリに配置してください。

### ダウンロード URL の取得

manifest ファイルを開きます：

```text
https://cdn.jsdelivr.net/gh/InscuraApp/inscura-jellyfin-plugin@release/manifest.json
```

その中で最新バージョンの `sourceUrl` をコピーし、対応する zip ファイルをダウンロードします。

現在公開されているファイルの URL 形式は次のようなものです：

```text
https://cdn.jsdelivr.net/gh/InscuraApp/inscura-jellyfin-plugin@release/releases/Inscura_[バージョン].zip
```

### プラグインファイルの配置

Jellyfinのデータディレクトリを見つけ、その中の`plugins`ディレクトリに移動します。一般的な場所は以下の通りです。お使いのシステムがこれと異なる場合は、Jellyfinコンソールに表示されるデータディレクトリ、またはデプロイ時の実際のマッピングパスを基準にしてください。

| 環境 | 一般的なプラグインディレクトリ |
| --- | --- |
| Linux パッケージインストール | `/var/lib/jellyfin/plugins/` |
| Docker | ホストマシンからコンテナ内の `/config/plugins/` ディレクトリへのマッピング |
| Windows サービスインストール | `%ProgramData%\Jellyfin\Server\plugins\` |
| Windows ポータブルまたはユーザーモード | Jellyfin データディレクトリ内の `plugins\` |

手動インストール手順：

1. Jellyfin を停止するか、コピー完了後に Jellyfin を再起動する準備をしてください。
2. `plugins` ディレクトリの下にバージョンディレクトリを作成します。例：

```text
plugins/Inscura_0.1.0.0/
```

3. zip ファイルを解凍し、その中のファイルを当該ディレクトリに配置します。
4. ディレクトリには、プラグイン DLL のみが必要です：

```text
Jellyfin.Plugin.Inscura.dll
```

5. Jellyfin プロセスがこのファイルに対して読み取り権限を持っていることを確認してください。
6. Jellyfin を再起動します。
7. コンソールのプラグインページにアクセスし、`Inscura` が読み込まれていることを確認します。

## Jellyfin でプラグインを有効にする

プラグインのインストール後、映画ライブラリで Inscura メタデータソースを有効にする必要があります。

1. Jellyfin コンソールにアクセスします。
2. メディアライブラリを開き、映画ライブラリを編集します。
3. メタデータダウンローダーで `Inscura` を有効にします。
4. `Inscura`を他の映画メタデータソースよりも上位に配置することをお勧めします。
5. 画像ダウンローダーで`Inscura`を有効にします。
6. メディアライブラリの設定を保存します。
7. プラグイン設定ページに移動し、Inscura API アドレスとアクセストークンを入力します。
8. 映画に対してメタデータの更新または識別を実行します。

初めて使用する際は、まず少数の映画を選んでメタデータを更新し、タイトル、出演者、カバー画像、背景画像が期待どおりであることを確認してから、データベースを一括更新することをお勧めします。

## プラグイン設定の説明

| 設定 | 説明 |
| --- | --- |
| Inscura API URL | Inscura のローカル API サービスのアドレス。Jellyfin と Inscura が同一のマシンにない場合は、LAN アドレスを入力する必要があります |
| API Token | ローカル API サービスでトークン認証を使用する場合に入力します。Inscura が認証なしに設定されている場合は、空欄のままにできます |
| Search result limit | 1回のマッチングごとにInscuraからリクエストする候補の数 |
| Request timeout | Inscura ローカル API へのリクエストのタイムアウト時間 |
| Enable movie metadata provider | 映画メタデータのスクレイピングを有効または無効にする |
| Enable image provider | 画像のインポートを有効または無効にする |
| Import YouTube trailers | 有効にすると、YouTubeのリモート予告編のみをインポートする |
| Inscuraのプレビュー画像をサムネイル候補として使用 | Inscuraのプレビュー画像をJellyfinのサムネイル候補として使用 |
| ギャラリー画像をバックドロップ候補として使用 | Inscuraのギャラリー画像をJellyfinのバックドロップ候補として使用 |

## 使用上の推奨事項

- 初めて使用する際は、まず少数の映画を更新し、マッチング結果が期待通りであることを確認してから、一括更新を行ってください。
- Jellyfin内で映画のタイトルが手動で変更されていた場合でも、プラグインはタイトルを唯一の基準とせず、実際のファイルパスとファイル名によるマッチングを優先します。
- Inscuraのサービスアドレスやトークンが変更された場合は、Jellyfinのプラグイン設定で同期更新を行い、その後メタデータを更新する必要があります。
- Inscuraのメディアライブラリがロックされているか、停止されている場合、Jellyfinはメタデータを読み取ることができません。
- YouTube以外の予告編については、プラグインによるインポートに依存せず、InscuraアプリのNFO機能を使用してJellyfinにエクスポートし、スキャンを行ってください。

## トラブルシューティング

### JellyfinでInscuraプラグインが表示されない

1. リポジトリからインストールした場合は、Jellyfin サーバーからリポジトリの URL にアクセスできることを確認してください。
2. 手動でインストールした場合は、プラグインファイルが Jellyfin のデータディレクトリ内の `plugins/Inscura_[バージョン]/` にあることを確認してください。
3. ディレクトリ内に `Jellyfin.Plugin.Inscura.dll` が存在することを確認してください。
4. Jellyfin プロセスがこのファイルを読み取る権限を持っていることを確認してください。
5. Jellyfin を再起動してください。
6. Jellyfin のウェブインターフェースを再度開き、コンソールのプラグインページで確認してください。

### Jellyfin でプラグインは表示されるが、メタデータがない場合

1. Jellyfin サーバーが稼働しているデバイスで、Inscura のヘルスチェック URL にアクセスしてください。
2. Inscura メディアライブラリが起動しており、ローカルインターフェースサービスが有効になっていることを確認してください。
3. プラグイン内の Inscura API URL が、誤った `127.0.0.1` になっていないことを確認してください。
4. インターフェースがトークン認証を使用している場合は、プラグインに正しいトークンが入力されていることを確認してください。
5. 映画ライブラリのメタデータダウンローダーで、`Inscura`が有効になっていることを確認してください。
6. 作品に対してメタデータの更新を実行し、更新モードで「すべてのメタデータを上書き」を選択してください。

### カバー画像、背景画像、またはキャストのアイコンが表示されない

1. Jellyfin サーバーが Inscura API URL にアクセスできることを確認してください。
2. インターフェースがトークン認証を使用している場合は、プラグインのトークンが正しいことを確認してください。
3. Inscura 内で、該当するメディアまたはキャストに利用可能な画像リソースが実際に存在することを確認してください。
4. 映画ライブラリの画像ダウンローダーで `Inscura` が有効になっていることを確認してください。
5. 当該作品のメタデータを更新し、必要に応じて既存の画像を置き換えるように選択してください。

### 予告編がインポートされない

1. プラグインの設定で「`Import YouTube trailers`」が有効になっていることを確認してください。
2. Inscura 内の該当する予告編の URL が YouTube の URL であることを確認してください。
3. 現在のプラグインでは、YouTube以外のローカルまたはオンラインの予告編はインポートされません。
4. YouTube以外の予告編をインポートする必要がある場合は、InscuraアプリのNFO機能を使用してエクスポートし、Jellyfinでスキャンしてください。

## プラグインのアップグレード

### プラグインリポジトリからのアップグレード

1. プラグインリポジトリの URL が引き続きアクセス可能であることを確認してください。
2. Jellyfin コンソールのプラグインページにアクセスしてください。
3. `Inscura` に利用可能な更新があるか確認してください。
4. 更新をインストールしてください。
5. Jellyfin を再起動してください。

### 手動でのアップグレード

1. Jellyfinを停止します。
2. 新バージョンのzipファイルをダウンロードします。
3. `plugins`ディレクトリの下に、新しいバージョン用のディレクトリ（例：`Inscura_0.1.1.0`）を作成します。
4. 新バージョンのzipファイルをそのディレクトリに解凍します。
5. 新しいバージョンが正常に読み込まれることが確認できるまで、旧バージョンのディレクトリは残しておきます。
6. Jellyfinを起動します。
7. 新しいバージョンが正常に動作していることを確認してから、旧バージョンのディレクトリを削除します。

アップグレード後、Jellyfinのウェブインターフェースに旧設定や旧メタデータが表示された場合は、まずブラウザのページを強制リロードし、その後、プラグインの設定を再度開くか、動画のメタデータを更新してください。
